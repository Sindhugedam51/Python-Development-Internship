# 📝 Task 1 - To-Do List Application

## 📌 Objective

Develop a command-line To-Do List application using Python that allows users to manage their daily tasks efficiently. The application enables users to add, view, mark as completed, and delete tasks through a simple menu-driven interface.

---

## 🚀 Features

- ➕ Add a new task
- 📋 View all tasks
- ✅ Mark a task as completed
- 🗑️ Delete a task
- 🔄 Menu-driven interface
- ⚠️ Handles invalid user input gracefully

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)

---

# 📊 Word Count Tool

## Description

This project analyzes a text file and provides useful statistics such as:

- Total Words
- Total Characters
- Total Lines
- Top 10 Most Frequent Words

## Features

- Reads a text file
- Counts words
- Counts lines
- Counts characters
- Displays top 10 frequent words
- Exception handling

## Technologies Used

- Python 3

## Concepts Used

- File Handling
- String Manipulation
- Collections (Counter)
- Exception Handling
- Functions

## How to Run

```bash
python word_count.py
```

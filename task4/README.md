# Task 4 - Basic Web Scraper

## Description

This Python application scrapes the latest news headlines from the BBC News website using the Requests and BeautifulSoup libraries.

## Features

- Fetches live webpage content
- Extracts news headlines
- Displays the first 10 headlines
- Handles connection errors gracefully

## Technologies Used

- Python
- Requests
- BeautifulSoup4

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python web_scraper.py
```

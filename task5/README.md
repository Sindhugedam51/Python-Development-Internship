# 💱 Currency Converter

## Description

This Python application converts currencies using live exchange rates fetched from an online API.

## Features

- Live exchange rates
- Supports multiple currencies
- User-friendly interface
- Handles invalid inputs
- Exception handling

## Technologies Used

- Python
- Requests API

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python currency_converter.py
```

## Example

```
From: INR
To: USD
Amount: 500

Result:
500 INR = 5.81 USD
```

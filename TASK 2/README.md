# 🎯 Guess the Number Game

![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python)
![Status](https://img.shields.io/badge/Status-Completed-success)
![Platform](https://img.shields.io/badge/CLI-Terminal-lightgrey)

A simple Python game where the computer randomly selects a number between **1 and 100**, and the player tries to guess it.

---

## Features

- Random number generation
- Unlimited guessing attempts
- Higher / Lower hints
- Attempt counter
- Input validation
- Play Again option

---

## Technologies Used

- Python 3
- random module
- Loops
- Conditional Statements
- Exception Handling

## Run the Project

Clone the repository

```bash
git clone https://github.com/yourusername/Guess-The-Number-Game.git
```

Go inside the folder

```bash
cd Guess-The-Number-Game
```

Run

```bash
python guess_number.py
```
